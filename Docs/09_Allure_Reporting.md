#  Enterprise Automation Framework

# 09 - Allure Reporting Guide


## Introduction

 Enterprise Automation Framework provides enterprise-grade reporting using Allure Report.

The reporting module is designed to provide complete execution visibility including:

- Test execution status
- Test steps
- Failure analysis
- Screenshots
- Videos
- Trace files
- Environment details
- Execution details
- Historical trends
- Self Healing evidence


The goal:

```
Every automation execution should provide enough information
to analyze failures without re-running tests.
```

# StepLogger


Location:


```
framework/reporting/

    StepLogger.js
```


Purpose:


Create readable execution steps.


---


# Without StepLogger


Report:


```
click()

fill()

expect()
```


Hard to understand.


---


# With StepLogger


Code:


```javascript
await StepLogger.step(

"Login with valid user",

async()=>{


    await loginPage.login();


});
```


Allure:


```
Login with valid user

       PASSED
```
